# online_tutor
 
An interactive, educational CLI tool built with Python and SymPy designed to help students master Algebra concepts. Unlike standard solvers that instantly spit out an answer, this tool acts as a digital tutor, breaking down equations step by step using fundamental algebraic properties to show students exactly how to solve problems.

This project serves as both an educational resource for students and a deep-dive exploration into symbolic mathematics and expression-tree manipulation using SymPy for myself.
